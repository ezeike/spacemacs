## Main View {#main-view}

j Jump to maildir folder
s Enter a mu find search string (see below)
C Compose a new email
bx Run bookmark x
; Switch context
U Update email and mu database
q Quit mu4e


## Message View {#message-view}

u - unmark
x - execute marks

C-j/k Move through messages without focusing
