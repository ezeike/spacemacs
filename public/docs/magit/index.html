    Shortcuts #

    KeysFunctionDescription
    SPC g s d dDiff between local and remote (type master in prompt)
    SPC g s b xDelete a branch
    SPC g bGit blameGit blame

    Adding existing project to gitlab #

    SPC g i git init

    > git remote add origin git@gitlab10:enielson/project.git > git push -u origin master
